Keyboard Shortcuts

Quickly navigate and manage River using shortcuts. Since standard web browsers cannot access your keyboard's hardware-level fn key alone, we map the fn modifier to the Alt (Option) key in the browser environment.

Quick Add Transaction Alt + T (fn+T)
Dashboard Alt + D (fn+D)
New Plan Alt + P (fn+P)
Ledger (Transactions) Alt + L (fn+L)
Statistics Alt + S (fn+S)
Settings Alt + G (fn+G)
